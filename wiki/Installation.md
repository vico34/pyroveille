# Installation

## HACS

1. Ouvrir HACS.
2. Aller dans `Integrations`.
3. Ajouter ce depot comme depot personnalise :

   `https://github.com/vico34/pyroveille`

4. Choisir la categorie `Integration`.
5. Installer `PyroVeille`.
6. Redemarrer Home Assistant.
7. Ajouter l'integration depuis `Parametres > Appareils et services`.

## Installation manuelle

Copier `custom_components/pyroveille` dans le dossier `custom_components` de Home Assistant, puis redemarrer Home Assistant.
